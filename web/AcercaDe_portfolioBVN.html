Para la realización de este proyecto se partió del siguiente prototipo, el cual fue realizado mediante Adobe InDesign CC.

  • • En él, se parte los siguientes conceptos:
    • · Una pagina web estática con 6 secciones
    • · Un estilo minimalista a escala de grises
    • · Elementos geométricos como elementos visuales decorativos ( Se descartan )
  • • Finalmente, evoluciona junto estos conceptos:
    • · Uso exclusivo de líneas y logos como elementos visuales decorativos
